How does it work?

For a target drug t:

  1. Find Morgan fingerprints of t and other drugs in the database {t′}
  2. Compare fingerprints to compute similarity indices {Stt′}
    • 0 ≤ Stt′ ≤ 1
    • Stt′ = 0 indicates no structural similarity
    • Stt′ = 1 indicates identical structures
  3. Choose a similarity threshold Smin
  4. Select drugs with Stt′ > Smin

For a category of drugs:

  1. Select approved drugs in the target category
  2. Set similarity threshold by clustering approved drugs
  3. Perform the target drug algorithm for each approved drug
